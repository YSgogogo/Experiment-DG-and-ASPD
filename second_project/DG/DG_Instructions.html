BLOCK 1: Instructions


Structure of the block

  • You will face 5 rounds of the game in this block.
  • Each round is equally likely to be selected for payment.
  • Thus, it is important for you to take each round seriously.

Structure of the round

  • In each round you assume a role of First Mover or Second Mover.
  • First Mover is the only active player in the round.
  • First Mover decides on what action to take: A or B.
  • First Moverâ€™s decision determines the payment for both First Mover and Second Mover.
  • An example below illustrates the payment structure:
First Mover payment Second Mover payment
First Mover choice A 40 tokens 45 tokens
B 50 tokens 20 tokens
  • If First Mover decides:
    • A, then First Mover receives 40 tokens and Second Mover receives 45 tokens
    • B, then First Mover receives 50 tokens and Second Mover 20 tokens.
  • In this block, once you have made a decision assuming the role of the First Mover you can proceed to the next round.
  • In each round you will be presented with the new payment structure within the same decision problem:
    • The only decision is done by the First Mover picking A or B.
    • Payments corresponding to each of actions vary.

Payments

  • The payments for each round are determined as follows.
  • You are randomly matched with another player in the room.
  • You are randomly assigned a role (First Mover or Second Mover).
  • The decision of the First Mover is implemented in the group.
  • One of the rounds will be selected for payment.
  • You will only see outcomes in the end of the experiment.
